Cronómetro¶

Acerca de¶

Cronómetro es el cronómetro por excelencia de Sugar, intuitivo y además fácil de usar, soporta las necesidades tanto de medición de tiempo simple y avanzada. Entre sus innovaciones, Cronómetro ofrece la posibilidad de gestionar varios temporizadores simultáneamente y para que sea más fácil de administrar tus relojes, puedes cambiar el nombre de todos ellos.

Dónde obtener StopWatch¶

Si bien la actividad Cronómetro no viene preinstalado en todas las distribuciones de software de OLPC, se puede descargar la página de la actividad

Uso¶

De hecho, la actividad Cronómetro es muy fácil de usar, pero si no entiendes alguna de las características de esta actividad, las referencias se enumeran a continuación:

  1. Selecciona el nombre de un temporizador para mantener tu trabajo organizado.

  2. Inicia / Pausa el cronómetro.

  3. Reinicia el cronómetro.

  4. Crear una marca de tiempo (en otros temporizadores, conocido como “Una marca”).

  5. EL visor del tiempo actual.

  6. Una lista de todos los tiempos que haz marcado.
